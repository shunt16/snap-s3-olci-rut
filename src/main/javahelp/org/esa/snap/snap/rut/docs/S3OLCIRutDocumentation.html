S3-OLCI-RUT

Sentinel-3 OLCI Radiometric Uncertainty Tool (S3-OLCI-RUT)

The S3-OLCI-RUT produces per-pixel uncertainty of ESA Sentinel-3 L1B OLCI products at a specified confidence level. The L1 uncertainty budget is based on a detailed L1 radiometric assessment and its combination based on the GUM guidelines. Flexibility as a SNAP processor provides opportunities to: specify confidence level, specify output format, select/deselect contributors.

Note: RUTv0.0 is still a prototype version of the software developed from Sentinel-2 hertiage (credit: Javier Gorrono (NPL))
